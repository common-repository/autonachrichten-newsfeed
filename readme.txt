== Description ==

UPDATE INFO:
Vielen Dank für die Zahlreichen Bugreports an wordpress@autonachrichten.de. Wir freuen uns über jeden Bugreport um unser Plugin weiter verbessern zu können.

eine kleine Demo ist unter: http://blog.mtosch.de/ zu sehen.
Je nach installiertem Theme, sieht auch das Widget in deinem Blog anders aus.

Was macht unser Plugin aus?

* Es werden ganz automatisch die Nachrichten aus der Automobil-Branche in deinem WordPress Blog angezeigt.
* Suchen Sie sich selbst aus, von welchen Herstellern Sie Nachrichten zeigen wollen.
* Entscheiden Sie welche Kategorien für Ihren Blog relevant sind.
* Keine Performance Einschnitte, durch Caching

Dies ist eine Beta Version unseres Plugins. Wir sind noch auf der Suche nach Verbesserungen und Anregungen um das Plugin für Autonachrichten. Falls du Ideen oder Wünsche hast, schreib uns eine Email an wordpress@autonachrichten.de

== Installation ==

Wie installiere ich das Plugin?

* Lade dir das Plugin unter WordPress.org (http://wordpress.org/extend/plugins/autonachrichten-newsfeed/) herunter.
* Aktive das Plugin in den Plugin Einstellungen.
* Im Menüpunkt Widget’s kannst du das Widget "autonachrichten" in die Sidebar ziehen. Jenachdem wo du es platzierst, erscheint es dann in deinem Blog

== Frequently Asked Questions ==

1. Dürfen die Bilder der Autos auf meinem Blog erscheinen?
Solange keine Urheberhinweise in unserem Wordpress Plugin verändert werden, können alle Inhalte die über das Plugin angezeigt werden auch angezeigt werden

2. Ist es Möglich auch eigene Inhalte über das Netzwerk von autonachrichten.de zu veröffentlichen
Ja, wir nehmen immer Inhalte auf, wenn Sie in unseren Kontext passen
